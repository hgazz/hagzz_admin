# Project Overview

Welcome to the documentation for your Laravel admin panel project.

## Modules

The admin panel includes the following modules:

-Auth:Authentication routes for admin login/logout.<br>
-Cities: Manages the creation, editing, and deletion of cities within the application.<br>
-Areas:  Handles the management of areas, including CRUD operations for area data.<br>
-Sports: Manage sports categories and related data.<br>
-Partners: Manage partners and associated locations.<br>
-Bookings: Booking and reservation handling.<br>
-Banners:  Manages banners used within the application, including creation, editing, and deletion.<br>
-Settings: Admin section for managing application settings, including configuration and preferences.<br>
-Notifications: Manages notifications sent to users, allowing the admin to view and configure notification settings.<br>
-FAQs: Provides a management interface for Frequently Asked Questions, allowing for CRUD operations.<br>
-Countries: Manages country data, including adding, editing, and deleting country information.<br>
-Profile: Admin interface for managing admin user profiles, including updating personal information.<br>
-Users:   Handles user management, including viewing, editing, and deleting user accounts, as well as exporting user data.<br>
-Trainings: Manages training sessions, including viewing, booking, and exporting training-related data.<br>
-Coaches: Admin interface for coach management.<br>
-Reports: Admin reporting system, including filters and exports.<br>
